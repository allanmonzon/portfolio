About

Work

The Market Place

An Ember.js app that connects clients with freelancers. I worked on this app with Jeff Coons. I built the client side of the app, which includes the ability to create, post and accept a freelancer for your job. I also built a chat room where the client and freelancer can discuss project details.

Used: Sass, Bourbon, Neat, Ember.js, Ember-Data, Firebase

Batch Maker

Batch Maker is an Ember.js app that lets you create recipes. From adding an image, setting prep time, cooking time, recipe type, adding steps, etc. You can edit and share recipes. You can also view public recipes. It uses Filepicker to upload images.

Used: Sass, Bourbon, Neat, Ember.js, Ember-Data, Firebase, Filepicker

Github Profile

Using Github's API, my task for this project was to recreate my github repositories page. I pull in the dynamic data and keep it up to date by making AJAX calls to Github's API. I used Underscore's templating feature to render the data into the HTML.

Used: Sass, JavaScript, jQuery, AJAX, Underscore.js
